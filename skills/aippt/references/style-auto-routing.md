# Style Auto Routing

Use this file in **Stage 0** when choosing a starting preset or explaining the recommendation.

## Quick routing

| Request signal | Prefer preset | Typical dimensions |
| --- | --- | --- |
| board / executive / management review | `business` or `minimal` | `clean + neutral + geometric + minimal/balanced` |
| product launch / SaaS / AI product / developer | `tech` | `grid + cool + geometric + balanced` |
| teaching / classroom / onboarding / workshop | `scientific` or `minimal` | `grid/clean + warm/cool + humanist/technical + balanced` |
| research / policy / industry analysis / thesis | `scientific` or `editorial-infographic` | `grid/editorial + cool + technical/editorial + dense` |
| marketing / brand / campaign / creative keynote | `creative` | `organic + vibrant + editorial + balanced` |

## Audience bias

- executive / board audiences -> lean `minimal` or lower-density `business`
- investor / technical buyer audiences -> lean `tech`
- academic / evaluator audiences -> lean `scientific`
- data-dense storytelling -> lean `editorial-infographic`

## Decision rule

Start with the scene pack preference if one exists. Only override it when the user provides explicit brand or stylistic direction.
