# Board Briefing

Use this scene pack before the generic workflow when the request is a new board or executive briefing.

## Defaults

- Audience bias: board members, executive leadership, supervisory committee
- Page budget: 10-18 pages, default 14
- Style preset: `editorial-infographic`
- Delivery default: `prompt_bundle_only`
- Story arc: `anchor -> proof -> proof -> bridge -> proof -> closing`

## Required sections

- executive summary
- performance snapshot
- key drivers
- risks and mitigations
- decision points
- next-quarter actions

## Evidence bias

- prefer KPI stacks, tables, dated charts, and explicit comparables
- avoid decorative slides unless they reset density strategically

## Review bias

- prioritize `density`, `chart_legibility`, `citation_visibility`

Read `references/outline-starter.md` for the default board sequence before resuming the main AIPPT stages.
