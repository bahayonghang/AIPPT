# Thesis Defense

Use this scene pack before the generic workflow when the request is a new thesis or research defense deck.

## Defaults

- Audience bias: committee members, faculty reviewers, research audience
- Page budget: 10-18 pages, default 14
- Style preset: `scientific`
- Delivery default: `prompt_bundle_only`
- Story arc: `anchor -> proof -> bridge -> proof -> closing`

## Required sections

- research question
- literature gap
- method
- results
- contribution
- limitations and future work

## Evidence bias

- claims should stay tied to thesis data and clearly scoped contributions
- charts and citations should be readable at defense-room distance

## Review bias

- prioritize `citation_visibility`, `chart_legibility`, `readability`

Read `references/outline-starter.md` for the default defense sequence before resuming the main AIPPT stages.
