# Company Intro

Use this scene pack before the generic workflow when the request is a new company introduction deck.

## Defaults

- Audience bias: prospects, partners, procurement stakeholders
- Page budget: 8-15 pages, default 12
- Style preset: `business`
- Delivery default: `prompt_bundle_only`
- Story arc: `anchor -> proof -> bridge -> proof -> closing`

## Required sections

- company snapshot
- problem framing
- solution overview
- proof points
- deployment or service model
- call to action

## Evidence bias

- prefer official website, product docs, release notes, first-party customer proof
- dated third-party comparisons only when they sharpen the buying decision

## Review bias

- prioritize `hierarchy`, `layout_balance`, `citation_visibility`
