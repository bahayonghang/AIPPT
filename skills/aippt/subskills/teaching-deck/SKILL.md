# Teaching Deck

Use this scene pack before the generic workflow when the request is a new lesson or training deck.

## Route when

- lecture slides / course slides / class slides
- training deck / workshop deck / instructional presentation
- lesson deck with learning goals, examples, and practice

## Defaults

- Audience bias: students, trainees, classroom participants
- Page budget: 10-24 pages, default 16
- Style preset: `scientific`
- Delivery default: `prompt_bundle_only`
- Story arc: `anchor -> bridge -> proof -> breathing -> proof -> closing`

## Required sections

- learning goals
- concept framing
- worked example
- practice or exercise
- summary

## Evidence bias

- optimize for comprehension and pacing before density
- citations are required only when the lesson depends on external factual claims

## Review bias

- prioritize `readability`, `hierarchy`, `layout_balance`

## Near misses

- manuals, notes, or worksheet-only outputs -> route away from this scene pack
- research talk or conference talk decks -> stay generic unless the goal is teaching
- single cover or single activity slide -> route away from this scene pack
